# Editorial Brain Locked Fields

Locked fields are decisions downstream agents must preserve.

| Stage | Locked Fields |
|---|---|
| Match Selector | selected_match, competition, kickoff_time, production_id |
| Story Hunter | story_angle, central_question, surprising_fact |
| Evidence Filter | evidence_points, evidence_strength_rating |
| Insight Engine | match_edge, uncertainty_level, insight_dashboard |

Downstream agents may request revision, but they may not quietly rewrite locked fields.
